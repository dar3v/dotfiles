# dotfiles
my main Arch Linux desktop environment

## info
- os: **arch linux**
- window manager: **river (wayland)**
- bar: **waybar**
- notification daemon: **swaync**
- display configuration: **kanshi**
- terminal emulator: **kitty**
- text editor: **nvim** + **lazyvim**
